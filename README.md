## 🎯 Problem Solutions

### 1. Best Time to Buy and Sell Stock (`BUY_SELL_STOCK_BEST_DAY.py`)
**Problem:** Find the maximum profit from a single stock transaction
- **Approach:** Kadane's algorithm / Dynamic Programming
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Key Learning:** Optimizing for minimum buy price and maximum profit

### 2. Maximum Product Subarray (`MAX_PRODUCT_SUBARRARY.py`)
**Problem:** Find contiguous subarray with maximum product
- **Approach:** Track both maximum and minimum products (handles negative numbers)
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Key Learning:** Handling negative values in product calculations

### 3. Power Function (`Pow(x,n)_Problem.py`)
**Problem:** Implement pow(x, n) without built-in functions
- **Approach:** Recursive / Binary Exponentiation
- **Time Complexity:** O(log n)
- **Space Complexity:** O(log n)
- **Key Learning:** Efficient exponentiation techniques

### 4. Remove Duplicates (`REMOVE_DUPLICATE.py`)
**Problem:** Remove duplicates from sorted array in-place
- **Approach:** Two-pointer technique
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Key Learning:** In-place array manipulation

### 5. Smaller Numbers Than Current (`SMALLER_NUMBERS.py`)
**Problem:** Count numbers smaller than current element
- **Approach:** Brute force (with planned optimizations)
- **Current Complexity:** O(n²)
- **Target Complexity:** O(n log n)
- **Key Learning:** Trade-offs between simplicity and efficiency
